# Jest

The repository contains some examples about unit testing with jest library:

### Packages used

- jest
- jest-babel
- jest-polyfill
- @babel/preset-env

## Examples:

- math.test.js

#### Description:

```
Testing some math functions like addition, substraction, multiplication, division.
```
- matchers.test,js

#### Description:

```
Using toEqual matcher.
```
- callback.test.js

#### Description:

```
Testing callbacks.
```
- arrays.test.ts

#### Description:

```
Testing arrays using toContain.
```
- numbers.test.ts

#### Description:

```
Testing number using toBeGreaterThan, toBeGreaterThanOrEqual. toBeLessThan, toBeLessThanOrEqual and toBeCloseTo.
```
